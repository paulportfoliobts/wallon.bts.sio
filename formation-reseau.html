🌐 Mise en place du service Kea DHCP

📡 Définition DHCP : Le protocole DHCP permet d'attribuer automatiquement des adresses IP et autres paramètres réseau aux appareils se connectant à un réseau, évitant ainsi la configuration manuelle.

🔧 Définition Kea : Kea est un service DHCP open-source et moderne, remplaçant ISC DHCP, conçu pour gérer dynamiquement les adresses IP.

🚀 Comment : Installé sur une VM Ubuntu, configuré directement avec Kea via ses fichiers de configuration JSON pour définir plages IP et baux dynamiques.

🎯 Pourquoi : Automatise l'attribution des IP dans un réseau, permettant à des postes clients d'en obtenir automatiquement
